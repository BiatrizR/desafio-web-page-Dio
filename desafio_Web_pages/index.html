Client X Server

Cliente servidor é um sistema de requisições e distribuição de dados. Na rede existem os fornecedores de recursos ou serviços a rede, que são os chamados servidores, e existem os requerentes dos recursos ou serviços, que são os clientes. O cliente solicita alguma função do servidor, sendo ele, o cliente, responsável por iniciar a comunicação com o servidor, enquanto o mesmo aguarda requisições de entrada.

Tipos de servidores:
  • Segurança - Firrewall e proxy
  • Arquivos - Imagens e docs
  • Streaming - Videos
  • E-mail - Dados de emails
  • Web - Requisições e acesso

Linguagens de marcação

Criada por Tim Berners-Lee, HTML significa HiperText Markup Language, ou Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website.

Com um documento com a extensão .html ou .htm., o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo.

este documento consiste em uma série de tags. As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, a fonte da letra, as cores, as quebras de linha e etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fachamento, como <tag> </tag>. Há também tags de estrutura única, como a tag <br/ > que realiza uma quebra de linha.
